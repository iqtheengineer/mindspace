# What Is Mindspace?

Mindspace is a free website template that helps you create your own personal corner of the internet.

Think:

- MySpace
- Tumblr
- Xanga
- Digital Gardens
- Personal Blogs

But updated for today.

Create:

- A thought archive
- A journal
- A scrapbook
- A gallery
- A portfolio
- A digital home


Change:

- Website title
- Tagline
- Hero video
- Colors
- Thoughts
- Quotes

No HTML required.
