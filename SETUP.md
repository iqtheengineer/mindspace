# Create Your Own Mindspace

No coding experience required.

If you can upload photos to Facebook, you can build a Mindspace.

---

# Step 1: Create a GitHub Account

Go to:

https://github.com

Create a free account.

Think of GitHub as the place where your website lives.

---

# Step 2: Create Your Own Copy

At the top of the Mindspace repository click:

Use this template

Then:

Create a new repository

Give it a name.

Examples:

field-notes

late-night-thoughts

life-in-progress

my-space

the-archive

---

# Step 3: Upload Your Video

Find:

hero.mp4

Delete it.

Upload your own video.

Ideas:

- Travel footage
- Camera roll memories
- Ocean footage
- City lights
- Creative projects
- Nature clips

Keep videos under 30 seconds.

Name it:

hero.mp4

---

# Step 4: Customize Your Site

Open:

config.js

You will see:

```javascript
title: "My Mindspace"
```

Replace with your own title.

Example:

```javascript
title: "Field Notes"
```

Change your tagline:

```javascript
tagline: "Things I don't want to forget."
```

---

# Step 5: Add Your Thoughts

In config.js find:

```javascript
thoughts: []
```

Replace the sample thoughts with your own.

Example:

```javascript
{
type: "thought",
text: "Most people are not afraid of failure. They are afraid of failing publicly."
}
```

---

# Step 6: Publish Your Website

Open:

Settings

Then:

Pages

Under Source choose:

Deploy from a branch

Branch:

main

Folder:

/ (root)

Click Save.

---

# Step 7: Wait

GitHub will publish your website.

This usually takes 1–2 minutes.

---

# Step 8: Find Your Website

Return to:

Settings → Pages

You'll see:

https://yourusername.github.io/repository-name/

That's your website.

Share it.

Bookmark it.

Make it yours.

---

# Updating Your Site

Want to add a thought?

Open:

config.js

Add a thought.

Commit changes.

GitHub automatically updates your website.

No servers.

No hosting bills.

No complicated setup.
